# Retail Sales Data Analysis (MySQL)

## Project Objective
Analyze retail sales data to understand revenue trends, customer behavior, and product performance using SQL.

## Tools Used
- MySQL
- SQL

## Database Schema
- customers
- products
- orders
- order_items
- payments

## Key Analysis
- Total revenue
- Revenue by category
- Top customers
- Monthly sales trend
- Payment method analysis

## Key Insights
- Electronics is the top revenue category
- UPI is the most used payment method
- Repeat customers contribute significantly to sales
